POO : statique

Le mot clé self fait référence à la classe où est déclarée la méthode.

Le mot clé static fait référence à la classe courante qui appelle la méthode.

class Mother {
    public static function testStatic() {
        echo self::class . ' ' . static::class . PHP_EOL;
    }
}


class Child extends Mother {
}

echo Mother::testStatic(); // Mother Mother
echo Child::testStatic();  // Mother Child
