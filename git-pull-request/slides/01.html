Jusque là avec Git et GitHub

&

git clone git@github.com:sebastien-raud/mon-depot.git

Récupération d'un dépôt distant

git add README.md

Ajout d'un fichier

git commit -m "first commit"

Prise en compte des modifications

git push

Envoi des modifications vers un serveur distant

git pull

Mise à jour d'un dépôt local depuis un serveur distant

git checkout -b nouvelle-branche

Création d'une branche locale

git checkout master git merge nouvelle-branche

Fusion d'une branche
