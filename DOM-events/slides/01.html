DOM - événements

Le DOM permet de gérer des événements en Javascript.

Un événement est une action qui se produit :

  • par exemple un clic de souris,
  • une touche préssée sur le clavier,
  • la souris qui passe sur un élément...

Javascript permet d'écouter un événement sur un élément et d'exécuter une action lorsque l'événement se réalise.
