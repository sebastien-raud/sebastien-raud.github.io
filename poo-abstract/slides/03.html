POO : statique

Le mot clé self fait référence à la classe où est déclarée la méthode :

class Date {
    public static $format = 'd/m/Y';

    public static function today() {
        return date(self::$format);
    }
    public static function tomorow() {
        $datetime = new DateTime('tomorrow');
        return $datetime->format(self::$format);
    }
}

echo 'aujourd\'hui : ' . Date::today() . PHP_EOL;
echo 'demain : ' . Date::tomorow() . PHP_EOL;
