Gestion de projet

Étape 2 : conception du projet

Cette étape permet de définir toutes les informations nécessaires au développement.

Elle contient plusieurs livrables :

  • spécifications fonctionnelles
    • description détaillée des fonctions du logiciel
    • cas d'usages / cas d'utilisation (use-cases)
    • scénarios utilisateurs (user stories)
    • croquis d'interfaces (wireframes)
  • spécifications techniques
    • description des données :
      • MCD : modèle conceptuel de données
      • MLD : modèle logique de données
      • MPD : modèle physique de données
    • architecture logicielle :
      • les outils
      • les librairies
      • l'organisation des fichiers
      • ...
  • architecture
    • système d'exploitation
    • serveur web
    • ...
  • méthodes
    • pair-programming
    • revue de code
    • ...
  • prototypage et/ou maquettage
