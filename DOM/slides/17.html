DOM

Manipulation : synthèse

Le DOM est la représentation du document HTML utilisable par Javascript.

Sélection

  • {document}.getElementById('identifiant') : retourne un élément à partir de son identifiant
  • {document | element}.getElementsByClassName('nom_classe') : retourne un tableau d'éléments à partir d'un nom de classe
  • {document | element}.getElementsByTagName('nom_balise') : retourne un tableau d'éléments à partir d'un nom de balise
  • {document | element}.querySelector('selecteur_css') : retourne un élément à partir d'un sélecteur CSS
  • {document | element}.querySelectorAll('selecteur_css') : retourne un tableau d'éléments à partir d'un sélecteur CSS

Création, ajout, suppression

  • document.createElement('nom_balise') : création d'un nouvel élément
  • element.append(elementToAppend) : ajoute un élément comme dernier enfant d'un autre élément
  • element.prepend(elementToPreprend) : ajoute un élément comme premier enfant d'un autre élément
  • element.remove() : supprime un élément

Modifier un élément

On utilise les attributs de l'élément Javascript, par exemple :

  • element.id : retourne ou modifie l'identifiant de l'élément
  • element.className : retourne ou modifie l'attibut "class" de l'élément
  • element.classList : permet de modifier les classes de l'élément
  • element.textContent : retourne ou modifie le contenu textuel de l'élément
  • element.innerHTML : retourne ou modifie le contenu HTML de l'élément
  • element.style : retourne ou modifie l'attribut "style" de l'élément
  • element.src : retourne ou modifie l'attribut "src" pour un élément image img
