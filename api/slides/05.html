XML / JSON

XML

  • Extensible Markup Language / langage de balisage extensible
  • créé par le W3C entre 1996 et 1999
  • très verbeux, peut être difficilement lisible
  • contraintes fortes sur la structure du document
  • pas de notion de tableaux

JSON

  • JavaScript Object Notation / Notation d'objet JavaScript
  • créé par Douglas Crockford entre 2002 et 2005
  • peu verbeux, assez lisible
  • souple et facile à apprendre
  • comprend les types de données généraux (chaînes de caractères, nombres, booléens, null, tableaux, objets)
  • pas de commentaires
