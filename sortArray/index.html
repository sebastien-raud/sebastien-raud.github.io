sortArray

Exercice

Le but de l'exercice est de créer une fonction sortArray capable de trier tous les éléments d'un tableau par ordre croissant.
On utilise une technique appelée tri à bulles pour réaliser le tri.

Sasir les données du tableau (séparées par des virgules) :

for (let i = 0; i < testedArray.length - 1; i++) {
  // On compare l'entrée courante avec l'entrée suivante.
  if (testedArray[i] > testedArray[i + 1]) {
    // Si l'entrée courante est supérieure à l'entrée suivante,
    // le tableau n'est pas bien ordonné, on retourne false.
    return false;
  }
}
// Si on arrive ici, c'est que le tableau est bien ordonné, on retourne true.
return true;
