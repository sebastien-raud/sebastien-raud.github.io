POO : abstract

abstract class Animal {
    protected $nom;
    protected $nombrePattes;

    public function infos() {
        echo $this->nom . ' a ' . $this->nombrePattes . ' pattes.';
    }
}
// ⚠ Impossible d'instancier un Animal
$animal = new Animal(); // Erreur

La classe Animal est abstraite.

Elle ne peut pas être instanciée. Il faut donc créer des classes enfants pour l'utiliser.

class Humain extends Animal {
    public function __construct() {
        $this->nombrePattes = 2;
    }
}

class Chien extends Animal {
    public function __construct() {
        $this->nombrePattes = 4;
    }
}
