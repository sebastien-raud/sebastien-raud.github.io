$renaud = [
    'prenom' => 'Renaud',
    'nom' => 'Séchan',
    'age' => 70,
    'estChanteur' => true
];

function biographie($personne) {
    $bio = $personne['prenom'] . ' ' . $personne['nom'] . 
           ' est âgé de ' . $personne['age'] . ' ans';

    if ($personne['estChanteur']) {
        $bio .= ' et c\'est un chanteur';
    }
    return $bio;
}

echo biographie($renaud);

Dans le code précédent, on "voit" bien que le tableau $renaud et la fonction biographie() ont un lien.

Ce lien n'est pas formel : on le "voit", on le "sent", on le "lit"... mais rien, dans le code, n'indique réellement ce lien.

On pourrait passer n'importe quel tableau associatif qui contient les bonnes clés à la fonction biographie().

C'est d'ailleurs un des objectifs d'une fonction : être réutilisable.

Mais le "lien" entre les différents tableaux éventuels — les données — et l'action réalisée par la fonction reste non formalisé.
