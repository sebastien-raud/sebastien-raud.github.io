Du MCD au MLD : les relations

̈Pour la relation CONTACT-CATÉGORIE on va donc :

  • créer un champ category_id dans la table contact
  • créer une clé étrangère du champ contact.category_id vers le champ category.id

Quel type de champ pour contact.category_id ?

Comme c'est une clé étrangère vers category.id, il doit avoir le même type : INT

Quelles spécificités pour contact.category_id ?

Comme le MCD nous indique une relation 0,1 de CONTACT vers CATÉGORIE

on peut ne pas avoir de CATÉGORIE dans un CONTACT, donc le champ peut être NULL (à cause du 0 dans la cardinalité)

Champ Type spécificités Description
category_id INT NULL, UNSIGNED, FOREIGN KEY (category.id) Identifiant de la catégorie à laquelle est rataché le contact
