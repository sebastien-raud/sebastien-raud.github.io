Stockage des données

Le localStorage

Attention : localStorage pour stocker des types de données complexes (objet, tableau, booléen, ...) il faut transformer la valeur en chaîne de caractères !

Pour ça, une façon de faire : JSON.stringify().

const data = {
    age: 31,
    color: 'orange',
};

const dataString = JSON.stringify(data);
localStorage.setItem('data', dataString);
