DOM

Manipulation : ajouter un élément

Partie d'un document HTML

<!DOCTYPE html>
<html>
    <head>
    <title>DOM</title>
    </head>
    <body>
    </body>
</html>

DOM

  • document
    • html
      • head
        • title
      • body

On peut ajout des éléments en Javascript après les avoir créés :

  • const elementH1 = document.createElement('h1');
  • // personnalisation de l'élément
  • elementH1.id = 'le-dom';
  • elementH1.classList.add('title');
  • // ajout de l'élément au body
  • document.querySelector('body').append(elementH1);

Note : append() ajoute à la fin, prepend() ajoute au début.
