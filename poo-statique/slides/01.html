POO : statique

Parfois on utilise des valeurs (propriétés) ou actions (méthodes) qui ne dépendent pas d'un contexte (objet).

C'est donc dommage d'instancier un objet pour utiliser ces propriétés ou méthodes :

class Date {
    public function today() {
        return date('d/m/Y');
    }
    public function tomorow() {
        $datetime = new DateTime('tomorrow');
        return $datetime->format('d/m/Y');
    }
}

$date = new Date(); // ??? mais pourquoi
echo 'aujourd\'hui : ' . $date->today() . PHP_EOL;
echo 'demain : ' . $date->tomorow() . PHP_EOL;
