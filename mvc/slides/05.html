MVC : Contrôleur

Le Contrôleur est responsable du traitement des actions de l'utilisateur.

Il est l'intermédiaire entre l'utilisateur, les données (Modèle) et le rendu (Vue).

C'est le chef d'orchestre de l'application :

  • il est lié à la requête de l'utilisateur
  • il sélectionne / manipule les données nécessaires à l'action
  • il effectue les traitements nécessaires
  • il appelle la vue et lui transmet les données
