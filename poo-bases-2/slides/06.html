class Personne {

    // définition des propriétés
    private $prenom;
    private $nom;
    private $age;
    private $estChanteur;

    // définition d'un constructeur avec un paramètre obligatoire et un optionnel
    public function __construct($paramPrenom, $paramNom = null) {
        $this->prenom = $paramPrenom;
        $this->nom = $paramNom;
    }

    // définition d'un méthode pour 
    // pouvoir modifier age
    public function setAge($paramAge) {
        // on peut tester par exemple si $paramAge 
        // est un entier ou un floattant
        if (is_int($paramAge) || is_float($paramAge)) {
            $this->age = $paramAge;
        }
    }

    // définition d'une méthode
    public function biographie() {
        $bio = $this->prenom . ' ' . $this->nom . 
                ' est âgé de ' . $this->age . ' ans';

        if ($this->estChanteur) {
            $bio .= ' et c\'est un chanteur';
        }
        return $bio;
    }
}

Ce type de méthode se nomme mutateur ou setter.

Par nommenclature ces méthodes sont nommées de la forme setNomPropriete.

Donc maintenant on peut modifier age avec le code suivant :

$renaud = new Personne('Renaud');
$renaud->setAge(70);

Dans notre exemple, ça ne règle pas le problème de la donnée. Je peux toujours écrire :

$renaud = new Personne('Renaud');
$renaud->setAge('Séchan');

Mais, dans la méthode, on peut y travailler...
