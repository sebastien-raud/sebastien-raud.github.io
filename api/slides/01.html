API ?

Application programming interface

Interface de programmation d’application / Interface de programmation applicative

Une API est une interface logicielle qui permet de « connecter » un logiciel ou un service à un autre logiciel ou service afin d'échanger des données et des fonctionnalités.

CNIL

Pourquoi des API ?

  • simplifient la conception et le développement d'applications
  • font gagner du temps de conception et développement
  • offrent plus de flexibilité car simplifient l'administration et l'utilisation
