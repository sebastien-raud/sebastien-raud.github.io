Un élément est un objet Javascript. Il est la représentation en Javascript d'une balise HTML.

Un élément contient :

  • des propriétés qui correspondent entre autres aux attributs des balises HTML ;
  • des méthodes qui permettent d'effectuer des actions sur l'élément.
