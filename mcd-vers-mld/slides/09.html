Du MCD au MLD : les relations

̈Et pour la relation CONTACT-TÉLÉPHONE ?

Le MCD indique que :

  • un CONTACT peut avoir 0 ou plusieurs TÉLÉPHONE
  • un TÉLÉPHONE est lié au moins 1 et maximum 1 CONTACT

Donc la clé étrangère doit être dans la table phone et référencer le champ contact.id.

Quelles spécificités pour phone.contact_id ?

Comme c'est une clé étrangère vers contact.id, il doit avoir le même type : INT

Par ailleurs le MCD indique qu'il y a toujours un CONTACT pour un TÉLÉPHONE, donc le champ est obligatoire, donc NOT NULL

Champ Type spécificités Description
contact_id INT NOT NULL, FOREIGN KEY (contact.id) Identifiant du contact auquel est rataché le téléphone
